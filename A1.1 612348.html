4. En el siguiente bloque cree la variable solo_H para separar la poblacin masculina mediante el uso de la condicin == "Male" en la columna de Sexo de la hoja de datos.

Despus se utiliza para el eje Y el nivel de obesidad como en el punto anterior y para el eje X se utiliza la altura, una variable cuantitativa que puede conseguirse de la base de datos.

Ya para graficar se utiliz la librera plt. Se utiliz tanto plt.figure para definir el tamao del grfico, la funcin plt.scatter grafica los datos indicados y con plt.show() nos muestra el grfico.

In[13]:
%matplotlib inline

import matplotlib.pyplot as plt

solo_H = data[data['Sexo'] == "Male"]
plt.figure(figsize = (10,4))
plt.scatter(x = solo_H.Estatura, y = solo_H.NivelDeObesidad, alpha = 0.1)
plt.show()
